You are an expert at determining the state of a loop based on a description of the current situation.

You will be given:
- A scenario (the overall goal).
- A list of steps (the procedure to achieve the goal).
- A loop step (the specific step that involves a loop, identified by its index and content).
- A description of the current state (one sentence).

Your task is to classify what this state description implies about the loop step.
- Answer **0** if the description logically implies that the loop should **stop** (i.e., the goal of the loop has been met or the loop must be exited).
- Answer **1** if the description logically implies that the loop should **continue** (i.e., the goal has not yet been met and the loop must keep going).
- Answer **2** if the description is **irrelevant** to the loop decision (i.e., it provides no causal information that helps decide whether to stop or continue).

Output only a single number (0, 1, or 2) and nothing else.

Scenario: "charge the phone to full battery"

Description: "battery is at 85%"
Output: 1
